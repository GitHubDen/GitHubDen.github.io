Project

Drone Shooter
Mini Game

Search and Destroy

This is a drone mini game. The project the implementation of OpenGL turned into a fun game. The application was built in visual studio using OpenGL and C/C++. The player is able to control the drones movement as well as camera views. Another drone is implemented with a simple AI to fly around the map. The player drone has the ability to fire a missile that can hit the AI drone, causing an explosion. The player will get points upon hits on the AI drone. If the player drone collides with the AI drone, the player drone will be destroyed and will need to respawn.

Key Features

Movement Control

Players can control the drone with movement keys as well as start spinning propellers.

Camera Control

Players can control camera view and angles using keys or mouse drags.

Ai Drone

An AI drone is your enemy, shoot them down to gain points!

Effects

Explosion will result from shooting down the AI drone.
