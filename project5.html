Project

PetStory

A 2d scroller mini game based off the MMORPG Maplestory. The use of this application is for the nostalgia of playing Maplestory as well for a music background player for old favourite Maplestory maps and some new ones. Maps with monsters are included for entertainment purposes and to provide engagement for those who may not have played the MMORPG Maplestory. Explore the world and enjoy the music.

This is a 2D game created using unity coded with C#. This game was created as a side project and was themed off a childhood game I used to play.
