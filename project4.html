Project

Artist Webpage
Development

Building from scratch

This is a webpage designed to demonstrate backend development with a few frontend elements. It was created using HTML5/CSS, PHP, JavaScript, and SQL. The project demonstrates the use of JavaScript to display and navigate images. Additionally PHP is used to create and handle user data by updating a database incorporated with SQL. A UI is implemented to demonstrate views for the database tables stored on the server.

Key Components

Layout

A simple layout accompanied with music.

Navigation System

The navigation system allows users to select different arts, artist, museums, and personalized pages.

New User Forms

A sign up form is implemented to store shoppers information into the database table.

Shopping Chart Form

A simple form desgined to give users the option of purchasing artworks.

Search Bar

A search bar allows the user to search the name of an artwork.

Search Results

Upon searching 'the' search results will present a clickable menue that will bring up the corresponding artwork.

Ui for Database

The UI is designed to allow easy acess to the backend databases to test functionality without external editors.

Editable Database Form

The art works avaible on the website can be edited directly from this UI. The ability to add, update, and delete specific data attributes from the database tables.
